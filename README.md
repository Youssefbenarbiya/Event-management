# Event Management Website

## Overview
The Event Management Website is a platform that allows users to create, manage, and participate in events seamlessly. It features functionalities such as event creation, registration, real-time updates, and search/filter options.

### Live Demo
Explore the website live at: [Event Management Website](https://event-dsi31.vercel.app/)

## Features
- **Event Creation**: Registered users can create events and provide necessary details such as date, time, location, and description.
- **User Registration**: Users can sign up or log in to manage events or book tickets.
- **Event Search and Filter**: Visitors can search for events by category, date, or location.
- **Real-time Updates**: Notifications and updates are available for the latest event changes.
- **Public Accessibility**: Non-authenticated users can view events but must log in for advanced features like booking or organizing events.

- **Deployment**: Vercel

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Youssefbenarbiya/Event-management.git
   ```
2. Navigate to the project directory:
   ```bash
   cd event-management
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm start
   ```

## Deployment
The project is deployed using Vercel. To deploy it locally:
1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy the `build` directory to your preferred hosting service.
